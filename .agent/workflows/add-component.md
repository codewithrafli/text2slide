# Add Component Workflow

1. Ask user which component they want to add (e.g., dialog, dropdown-menu, toast).

2. Run the shadcn add command:

```bash
npx shadcn@latest add [component-name] -y
```

3. Report which files were created.

4. Show a brief example of how to use the component.
